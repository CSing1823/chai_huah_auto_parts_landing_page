# Chai Huah Auto Parts & Service - Landing Page

A modern, professional, and responsive landing page for **Chai Huah Auto Parts & Service**, an automotive workshop and spare parts supplier located in Banting, Selangor, Malaysia.

## 📋 Project Overview

This is a **customer-facing business website** built with modern web technologies to showcase professional automotive repair and maintenance services. The website is fully responsive, SEO-optimized, and designed with a modern UI following Malaysian automotive workshop standards.

## 🎨 Design Features

- **Professional Theme**: Dark gray, red, and white color scheme
- **Responsive Design**: Fully responsive on mobile, tablet, and desktop
- **Modern UI**: Clean, contemporary design with smooth animations
- **Bootstrap 5**: Built on Bootstrap 5 framework for consistency
- **Bootstrap Icons**: Professional icon set throughout
- **Smooth Animations**: Fade-in, slide-in, and scale animations
- **Sticky Navigation**: Fixed navbar that stays at the top while scrolling
- **Floating WhatsApp Button**: Easy customer contact via WhatsApp
- **Google Maps Integration**: Embedded map showing business location
- **Contact Form**: Functional contact form with validation

## 📝 How to Customize

### 1. **Change Business Information**
Edit these in `index.html`:
- Company name: Search for "Chai Huah Auto"
- Phone number: `+603-6155-5555` and `+6012-3456-789`
- Address: "No. 12 & 14, Jalan Emas 12..."
- Email: `info@chaihuahauto.com`
- Business hours
- Facebook page link

### 2. **Update Colors**
Edit CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #dc3545;      /* Change this for main color */
    --dark-gray: #2c3e50;
    --light-gray: #ecf0f1;
    --white: #ffffff;
}
```

### 3. **Add Images**
Replace placeholder images in `images/` folder:
- `hero-car.jpg` - Hero section background
- `workshop.jpg` - About section
- `gallery1.jpg` through `gallery6.jpg` - Gallery section

Note: Onerror attributes fallback to SVG placeholders if images are missing.

### 4. **Update Services**
Edit the Services section in `index.html` to add or remove services.

### 5. **Modify Testimonials**
Edit the Testimonials section with real customer reviews.

### 6. **Update WhatsApp Number**
Replace `60123456789` with actual WhatsApp business number:
- In HTML: Search for `wa.me/60123456789`
- In JavaScript: No hardcoded numbers in JS

### 7. **Update Google Maps**
Replace the Google Maps embed URL with actual location:
```html
<iframe src="https://www.google.com/maps/embed?pb=...">
```
